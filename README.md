# Climbing Training Plan and Diary
This is an android application designed for climbers who are looking for a training plan. It uses an AI backend to generate said plans, while connecting to a Firebase database in the cloud.

## Features
  - AI powered training plan generation
  - Exercise timing
  - Exercise tracking and logging
  - Statstical tracking
  - Gamification
  - Authentication system

## Development Enviroment
The project was developed using Android Studio, as such further devlopement should take place in a similar or the same enviroment.

Android SDK:
  - Compile SDK: 34
  - Minimum SDK: 24
  - Target SDK: 34

A live internet connection is needed for connection to the Firebase Firestore database.

## Project Structure
  - \app\src\main\java\com\example\hsp | location of backend logic 
  - \app\src\main\res | location of front end logic
  - \generator-2 | location of AI model and translator

## Dependencies
 - Firebase | Database and authentication systems
 - Google Play Services: For Auth integration.
 - Material Design: For UI components
